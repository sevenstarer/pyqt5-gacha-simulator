# PyQt5 抽卡模拟游戏（课程大作业）
基于Python+PyQt5开发，复刻FGO抽卡概率与软硬保底机制，包含抽卡、图鉴、数据统计、存档重置功能。

## 功能模块
1. 抽卡模块：单抽/十连，85%普通/12%稀有/2.5%史诗/0.5%传说，十连保底稀有
2. 保底机制：90抽软保底传说概率翻倍，100抽硬保底必出传说
3. 数据统计：总抽数、各品质出货统计、柱状图可视化
4. 卡牌图鉴：解锁卡牌展示、品质筛选、详情弹窗
5. 辅助功能：初始1000抽额度、JSON存档、一键重置、预留多卡池接口
